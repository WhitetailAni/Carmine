# Carmine
A simple CTA bus tracker app for your menu bar. macOS 11.0+

CTA Bus Tracker API key is required. Apply for one at:
[[https://www.transitchicago.com/developers/traintrackerapply/](https://www.ctabustracker.com/dev-account)]([https://www.transitchicago.com/developers/traintrackerapply/](https://www.ctabustracker.com/dev-account))

Once it is supplied, simply build and run the app.
